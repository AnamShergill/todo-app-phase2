# Project & Agent Coordination Skill

## Overview
Coordinate tasks across multiple agents (spec architect, frontend, backend, QA). Ensure features are implemented according to project phases and specifications, and communicate changes effectively.

## Responsibilities
- Coordinate tasks across multiple development agents
- Ensure alignment with project specifications and phases
- Facilitate communication between different development roles
- Track progress across the project lifecycle
- Manage dependencies between different components

## Best Practices
- Maintain clear communication channels between agents
- Follow established project workflows and phases
- Document decisions and changes clearly
- Ensure all agents have necessary context
- Track and resolve inter-agent dependencies efficiently
