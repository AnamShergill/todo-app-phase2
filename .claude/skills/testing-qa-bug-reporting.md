# Testing, QA & Bug Reporting Skill

## Overview
Conduct thorough testing of frontend, backend, and API endpoints. Ensure security, usability, responsiveness, and spec compliance.

## Responsibilities
- Write and execute comprehensive test suites
- Perform security testing on all components
- Test usability and responsiveness across devices
- Verify compliance with specifications
- Document and report bugs effectively

## Best Practices
- Implement automated testing where possible
- Follow security best practices during testing
- Test across multiple browsers and devices
- Create detailed bug reports with reproduction steps
- Verify fixes before closing issues
