# Spec-Driven Development Skill

## Overview
Write, maintain, and interpret structured specifications using Spec-Kit Plus. Ensure all features, API endpoints, and database schemas are accurately described before implementation.

## Responsibilities
- Write clear, testable specifications for all features
- Maintain specifications throughout the development lifecycle
- Interpret specifications accurately during implementation
- Ensure all API endpoints are properly documented in specs
- Verify database schemas match specifications before implementation

## Best Practices
- Follow Spec-Kit Plus conventions strictly
- Keep specifications detailed but concise
- Include acceptance criteria in all specs
- Validate specs before implementation begins
- Update specs when requirements change
