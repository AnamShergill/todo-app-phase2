# Backend Development with FastAPI & SQLModel Skill

## Overview
Build scalable backend services using Python, FastAPI, and SQLModel ORM. Ensure secure user-specific data operations and database integration.

## Responsibilities
- Develop backend services using FastAPI framework
- Implement database models using SQLModel ORM
- Create secure user-specific data operations
- Integrate with PostgreSQL database (Neon)
- Implement proper error handling and logging

## Best Practices
- Follow FastAPI best practices for performance and documentation
- Use SQLModel for efficient database operations
- Implement proper validation and serialization
- Ensure secure access controls for user data
- Follow Python coding standards (PEP 8)
