# User Authentication & JWT Security Skill

## Overview
Implement authentication flows using Better Auth. Manage JWT issuance, verification, user isolation, and token expiry for secure API access.

## Responsibilities
- Implement authentication flows using Better Auth
- Handle JWT token issuance and verification
- Ensure proper user isolation in multi-user environments
- Manage token expiration and refresh mechanisms
- Implement secure password handling and storage

## Best Practices
- Follow JWT standards and security best practices
- Use strong encryption for sensitive data
- Implement proper token revocation mechanisms
- Secure all API endpoints that require authentication
- Handle token refresh securely and efficiently
