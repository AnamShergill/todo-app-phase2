# UI/UX Design & Theme Development Skill

## Overview
Create visually appealing, user-friendly, and consistent interfaces. Define color schemes, typography, spacing, responsive layouts, and component states.

## Responsibilities
- Design intuitive and user-friendly interfaces
- Create consistent color schemes and design systems
- Define typography and spacing guidelines
- Implement responsive layouts for all screen sizes
- Design component states (hover, active, disabled, etc.)

## Best Practices
- Follow modern UI/UX design principles
- Ensure accessibility standards are met
- Create consistent design language across the application
- Design for different device sizes and orientations
- Consider user journey and usability in all designs
