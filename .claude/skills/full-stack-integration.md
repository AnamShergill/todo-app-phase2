# Full-Stack Integration Skill

## Overview
Connect frontend and backend seamlessly, ensuring API responses, authentication, and user interactions work end-to-end. Debug integration issues and optimize data flow.

## Responsibilities
- Integrate frontend with backend API endpoints
- Ensure proper authentication flow between frontend and backend
- Debug integration issues across the stack
- Optimize data flow between client and server
- Test end-to-end functionality

## Best Practices
- Implement proper error handling across the stack
- Ensure consistent data formats between frontend and backend
- Test API integration thoroughly
- Monitor performance across the full stack
- Document integration points clearly
