# REST API Design & Implementation Skill

## Overview
Design and implement RESTful APIs with CRUD operations, route validation, error handling, and proper HTTP methods.

## Responsibilities
- Design RESTful API endpoints following industry best practices
- Implement CRUD operations for all resources
- Add proper route validation and request/response schema validation
- Implement comprehensive error handling and appropriate HTTP status codes
- Ensure proper use of HTTP methods (GET, POST, PUT, PATCH, DELETE)

## Best Practices
- Follow REST conventions consistently
- Use proper HTTP status codes (200, 201, 400, 401, 403, 404, 500, etc.)
- Implement rate limiting where appropriate
- Document API endpoints with clear request/response examples
- Include proper content negotiation (JSON, etc.)
