# Frontend Development with Next.js Skill

## Overview
Build responsive, interactive, and accessible user interfaces using Next.js (App Router), TypeScript, and Tailwind CSS. Integrate frontend with backend API endpoints using a centralized API client.

## Responsibilities
- Build responsive UI components using Next.js App Router
- Implement TypeScript type safety throughout the frontend
- Style interfaces using Tailwind CSS
- Create a centralized API client for backend communication
- Ensure accessibility standards are met

## Best Practices
- Follow Next.js App Router conventions
- Use TypeScript for type safety and better developer experience
- Implement proper component composition patterns
- Optimize for performance and SEO
- Ensure responsive design across all device sizes
