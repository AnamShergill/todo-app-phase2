# Database Design & Management (PostgreSQL / Neon) Skill

## Overview
Design relational schemas, indexes, and constraints in Neon Serverless PostgreSQL. Implement persistent storage for multi-user task data.

## Responsibilities
- Design relational database schemas for the application
- Create proper indexes for optimized queries
- Implement database constraints for data integrity
- Set up Neon Serverless PostgreSQL connection
- Manage multi-user data isolation

## Best Practices
- Follow PostgreSQL best practices for schema design
- Use proper indexing strategies for performance
- Implement proper normalization to reduce redundancy
- Ensure data security and user isolation
- Plan for scalability and performance optimization
